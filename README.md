<h2 align="center">ü§î Where to start?</h2> 

1. Install pipenv
   ```
   pip install --user pipenv
   ```
2. Install packages for correct operation
   ```
   pipenv install
   ```
3. Run Flask application
   ```
   pipenv run flask run
   ```
